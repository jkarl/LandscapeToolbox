What is Shiny?

Shiny is a web application framework for the R statistical software app that provides an easy way to create statistical and analytical tools that can run over the web. Shiny apps are developed in R on a desktop PC and is then hosted on a web server that is running R and Shiny. Shiny allows us to make sophisticated data analysis tools avaiable over the web to people who do not have access to statistics software apps. For more information on Shiny, check out R Studio's Shiny site.
